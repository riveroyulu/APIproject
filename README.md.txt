
# Tema: Carros

# Concessionária de Carros

Descrição do Projeto:

Projeto de sistema para o dono da concessionária, 
no qual ele vai poder cadastrar os modelos que vão ser 
oferecidos ao publico, com a descrição, características, 
preço, disponibilidade.

# Estórias de Usuário

- Eu como Administrador da concessionaria, gostaria de inserir e cadastrar novos carros (preço, modelo, descrição e ano)
- Eu como Administrador gostaria listar os modelos de carros para mim e para o usuario
- Eu como Administrador quero poder deletar carros que a concessionária não ira vender mais
- Eu como Administrador da concessionaria desejo atualizar os preços e as caraterísticas dos modelos oferecidos
- Eu como Administrador e o usuário teremos visualização se o carro escolhido esta disponível ou não.
